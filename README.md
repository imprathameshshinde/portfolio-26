# Portfolio Setup

1. Put your profile photo in `assets/prathamesh-photo.jpg`.
2. Put your resume PDF in root as `Prathamesh_Shinde_CV.pdf`.
3. Open `index.html` in browser to preview.

## Publish (GitHub Pages)

1. Create a new GitHub repo.
2. Upload all project files.
3. Go to repo `Settings` -> `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main` and folder `/ (root)`.
6. Save and wait 1-2 minutes.
7. Your link will be: `https://<your-github-username>.github.io/<repo-name>/`

## Add portfolio to LinkedIn

1. Open LinkedIn profile.
2. Click pencil icon in Intro section.
3. Find `Website` field and choose `Other`.
4. Paste your portfolio URL.
5. Set label like `Portfolio` or `Personal Website`.
6. Save.

Optional:
- Add the same URL in `Featured` section -> `Add link`.
